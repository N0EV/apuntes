# Guía de Contribución (Contributing Guidelines)

¡Gracias por el interés en mejorar este repositorio de apuntes! Al ser un espacio de estudio de carácter estrictamente personal, las contribuciones están limitadas para mantener la coherencia del material.

## ¿Cómo puedes ayudar?

No se aceptarán cambios directos en el contenido de las notas, pero eres más que bienvenido a colaborar de las siguientes formas:

1. **Reportar Erratas o Errores:** Si encuentras una falta de ortografía, un concepto técnico erróneo o un comando de Cisco mal escrito, abre una **Issue** pública explicándolo.
2. **Sugerir Mejoras:** Si crees que un concepto de ciberseguridad se puede explicar de forma más clara o quieres proponer un recurso complementario, puedes abrir una **Issue** de sugerencia.
3. **Enlaces Rotos:** Si ves que algún enlace del índice no funciona, avísame para que pueda corregirlo.

## Reglas importantes

Para mantener el repositorio seguro y legal:
* **No envíes Pull Requests con material oficial:** Cualquier intento de subir PDFs, exámenes o contenido protegido por derechos de autor de Cisco u otra organización será rechazado inmediatamente.
* **Sé constructivo:** Mantén siempre un tono educado y profesional en los comentarios de las Issues.
