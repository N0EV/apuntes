# SQL

Este repositorio contiene mis apuntes personales, resúmenes y ejercicios prácticos de **Structured Query Language (SQL)**. El objetivo de este espacio es servir como guía de referencia rápida y estructurada para dominar el diseño, la manipulación y la optimización de bases de datos relacionales.

## Qué encontrarás aquí
* **Teoría clara**: Conceptos fundamentales explicados sin rodeos.
* **Código listo para usar**: Ejemplos sintácticos limpios y listos para copiar.
* **Buenas prácticas**: Estándares de la industria para escribir consultas eficientes.
