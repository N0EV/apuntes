# Apuntes de Tecnologías Cisco®

Este directorio principal está dedicado a centralizar mis notas de estudio, resúmenes de arquitectura y prácticas de laboratorio personales sobre tecnologías de red y seguridad. 

El objetivo de este espacio es puramente didáctico, sirviendo como guía de aprendizaje y preparación técnica individual.

## Descargo de Responsabilidad y Cumplimiento de Políticas (NDA)

Para garantizar el respeto a la propiedad intelectual y el cumplimiento estricto de las políticas de certificación y academia:

* **Contenido 100% propio:** Toda la documentación, resúmenes conceptuales y esquemas de red han sido redactados y estructurados por mí, utilizando mis propias palabras y lógica técnica.
* **Sin material confidencial:** En este repositorio **no se aloja, comparte ni distribuye** material oficial de terceros, textos literales de currículos, imágenes ni laboratorios oficiales protegidos por derechos de autor.
* **Política Anti-Dumps / Exámenes:** Está estrictamente prohibido y ausente cualquier contenido relacionado con preguntas reales de exámenes de certificación, cuestionarios oficiales o soluciones directas a evaluaciones de academias privadas.
* **Laboratorios independientes:** Las topologías de red publicadas aquí son diseños propios creados desde cero para afianzar conceptos, sin replicar archivos o plantillas oficiales protegidas.
