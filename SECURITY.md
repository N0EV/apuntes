# Security Policy

## Reporting a Vulnerability

Si detectas cualquier problema de seguridad, filtración accidental de datos confidenciales o contenido que consideres que infringe derechos de autor en cualquiera de las secciones o estados de estos apuntes:

1. **Utiliza el reporte privado:** Por favor, ve a la pestaña **"Security"** de este repositorio en GitHub y haz clic en **"Report a vulnerability"** para enviar un informe de forma 100% privada.
2. **No abras una Issue pública:** Evita exponer el problema de forma abierta en el tablón público para mantener la privacidad y seguridad del repositorio.
3. **Acción inmediata:** Revisaré el informe privado lo antes posible. Si el reporte es correcto, el contenido afectado será corregido, anonimizado o eliminado de inmediato.
